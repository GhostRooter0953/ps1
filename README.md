# ps1

Simple role for Ansible, which installs ps1

## Usage (example)

```yaml
    - role: ps1
```

## Available parameters

| Param | Default | Description |
| -------- | -------- | -------- |
| `ps1_setup` | `full` | Setup mode. |
| `ps1_packages`| `[openssh-server, screen]` | Required packages |
